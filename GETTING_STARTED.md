## ğŸ’¡ Key Concepts

### Clamping Record States
```
New â†’ Pending â†’ Accepted â†’ Approved â†’ Released (or Cancelled)

New:      Just created
Pending:  Waiting for enforcer
Accepted: Enforcer will do it
Approved: Admin approved, ready
Released: Vehicle unclamped
```

### Payment States
```
Pending â†’ Completed (or Cancelled)

Pending:   Waiting for payment
Completed: Payment received
Cancelled: No longer needed
```

### User States
```
Pending â†’ Approved (or Rejected)

Pending:  Waiting for approval
Approved: Can login and use system
Rejected: Cannot login
Inactive: Disabled by admin
```
